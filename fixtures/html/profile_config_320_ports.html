VLAN Support
Enabling VLAN Support allows you to override existing VLAN schemas with the use of Array tags.
Would you like to enable VLAN support?
You will now be able to create VLAN IDs and Array tags for SSIDs (under the SSID or Policies tile) and User Groups (under the Policies tile) and assign them to Arrays (under the Arrays tab).
